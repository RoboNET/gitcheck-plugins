# gh ssh-key and gh gpg-key blocked by permissions deny

## Preconditions

- Agent container is running with baked `settings.json`
- `permissions.deny` includes `Bash(gh ssh-key*)`, `Bash(gh gpg-key*)`
- Claude Code session is active

## Steps

1. **Action:** Claude attempts `gh ssh-key add ~/.ssh/id_rsa.pub`.
   **Expected:** Blocked by permissions deny. Command not executed.

2. **Action:** Claude attempts `gh ssh-key list`.
   **Expected:** Blocked. Even read-only SSH key operations are denied.

3. **Action:** Claude attempts `gh gpg-key add key.asc`.
   **Expected:** Blocked by permissions deny.

4. **Action:** Claude attempts `gh gpg-key list`.
   **Expected:** Blocked.

## Postconditions

- No SSH or GPG key operations were performed
- Agent cannot enumerate or modify the user's cryptographic keys
