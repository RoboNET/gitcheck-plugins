# COMMENT_BODY and COMMENT_AUTHOR substituted in action template

## Preconditions

- `gitcheck` CLI is installed
- Plugin.yaml contains a custom rule:
  ```yaml
  - type: pr.commented
    labels: ["spec:*"]
    action: "/spec review --author {COMMENT_AUTHOR} --body {COMMENT_BODY}"
  ```
- A `pr.commented` event is emitted with `CommentBody: "Please fix the typo in section 3"`, `CommentAuthor: "reviewer42"`

## Steps

1. **Action:** EventRouter processes the `pr.commented` event and resolves the action template.
   **Expected:** The resolved action is `/spec review --author reviewer42 --body Please fix the typo in section 3`.

2. **Action:** Verify the resolved action is passed to the agent session.
   **Expected:** Claude Code receives the prompt with the fully substituted command.

## Postconditions

- `{COMMENT_BODY}` replaced with actual comment text
- `{COMMENT_AUTHOR}` replaced with actual author login
- No unresolved template variables remain in the action string
